Atributos e Métodos

Atributos são as características ou propriedades de um objeto. Eles representam o estado do objeto. No exemplo da classe Carro, modelo, cor e ano são atributos. São como as variáveis que armazenam os dados de um objeto.

Métodos são as ações ou operações que um objeto pode realizar. Eles definem o comportamento do objeto. No exemplo da classe Carro, exibirInformacoes() é um método. São como as funções que um objeto pode executar.

A palavra-chave this é usada dentro de um método para referenciar o próprio objeto. Ela permite acessar os atributos do objeto dentro do método.

Exemplo de código

                    
                        class Carro {

                        String modelo;
                        String cor;
                        int ano;

                        // Método construtor (inicializa os atributos do objeto)
                        Carro(String modelo, String cor, int ano) {
                        this.modelo = modelo;
                        this.cor = cor;
                        this.ano = ano;
                        } 

                        // Método para acelerar o carro
                        void acelerar() {
                        System.out.println("O carro " + modelo + " está acelerando!");
                        } 
                        
                        // Método para obter a cor do carro
                        String obterCor() {
                        return cor;
                            }
                        }

                        public class ExemploAtributosMetodos {

                        public static void main(String[] args) {

                        Carro meuCarro = new Carro("Sedan", "Prata", 2022);

                        meuCarro.acelerar();
                        
                        // Saída: O carro Sedan está acelerando!
                        
                        System.out.println("Cor do carro: " + meuCarro.obterCor());

                        // Saída: Cor do carro: Prata

                            }
                        }
