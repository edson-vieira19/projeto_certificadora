Encapsulamento

Encapsulamentoé um dos pilares da POO que consiste em esconder o estado interno de um objeto e restringir o acesso direto aos seus atributos. Isso é feito utilizando os modificadores de acesso.

Os modificadores de acesso definem a visibilidade de atributos e métodos:

  • public: Acesso de qualquer lugar.
  • private: Acesso apenas dentro da própria classe.
  • protected: Acesso dentro da própria classe, subclasses e classes no mesmo pacote (diretório).

Para acessar atributos private, utilizamos métodos getters (para obter o valor) e setters (para modificar o valor). Isso permite controlar como os dados são acessados e modificados, adicionando lógica de validação, se necessário.

Vantagens do Encapsulamento:


Segurança: Protege os dados do objeto de modificações indevidas.

Manutenção: Facilita a alteração interna da classe sem afetar o código que a utiliza

Flexibilidade: Permite controlar o acesso e a modificação dos dados de forma precisa.


Exemplo de Código:

                    
                        class ContaBancaria {

                        private int numeroConta;
                        private String titular;
                        private double saldo;
                        
                        // Construtor
                        public ContaBancaria(int numeroConta, String titular, double saldo) {
                        this.numeroConta = numeroConta;
                        this.titular = titular;
                        this.saldo = saldo;
                        } 
                        
                        // Getters
                        public int getNumeroConta() {
                        return numeroConta;
                        } 
                        public String getTitular() {
                        return titular;
                        }
                         public double getSaldo() {
                        return saldo;
                        }

                        // Setter (com validação)
                        public void setSaldo(double novoSaldo) {
                        if (novoSaldo >= 0) {
                        saldo = novoSaldo;
                        } else {
                        System.out.println("Saldo inválido.");
                            }
                        }
                        
                        public void depositar(double valor) {
                        saldo += valor;

                        System.out.println("Depósito de " + valor + " realizado. Novo saldo: " + saldo);
                        }

                        public void sacar(double valor) {
                        if (valor <= saldo) {
                             saldo -=valor;
                             System.out.println("Saque de " + valor + " realizado. Novo saldo: " + saldo);
                        } else {
                        System.out.println(" Saldo insuficiente.");
                             } 
                        } 
                    }
                    
                    public class ExemploEncapsulamento { 
                        
                        public static void main(String[] args) {
                            
                            ContaBancaria conta=new ContaBancaria(111222, "Carlos Drummond" , 2000.0);

                            System.out.println("Titular: " + conta.getTitular()); // Saída: Titular: Carlos Drummond

                        conta.setSaldo(-100.0); // Saída: Saldo inválido.

                        conta.setSaldo(2500.0);

                        System.out.println(" Saldo: " + conta.getSaldo()); // Saída: Saldo: 2500.0
                        }
                    }
