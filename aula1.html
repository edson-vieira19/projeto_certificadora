Introdução a POO

A Programação Orientada a Objetos, ou POO, é uma forma de escrever códigos que imita o mundo real.

Nela, a gente organiza o programa em “objetos”, que são como caixinhas que guardam informações e sabem fazer algumas tarefas. Por exemplo, se estivermos criando um jogo com carros, cada carro pode ser um objeto com suas próprias cores, velocidade e funções como acelerar ou frear.

Esses objetos são criados a partir de algo chamado classe, que é como um molde. A classe define o que todos os objetos daquele tipo vão ter e fazer. Os objetos podem ser criados no codigo a medida que precisarmos deles. A principal vantagem da POO é justamente essa aproximação com o mundo real, atraves dos objetos, podemos assim ter um código organizado e fácil de entender.

A POO se baseia em quatro ideias principais, chamadas de pilares: abstração, encapsulamento, herança e polimorfismo. Durante o decorrer do curso, vamos utilizar a linguagem Java para aprender e aplicar esses conceitos na prática, já que ela é uma das mais usadas quando se trata de programação orientada a objetos.
