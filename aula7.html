Abstração

Abstração é o processo de simplificar a complexidade, modelando as classes de acordo com os aspectos relevantes do problema, ignorando detalhes desnecessários. Ela permite que nos concentremos no que é essencial. Em Java, a abstração é alcançada através de classes abstratas e interfaces.

Classes Abstratas: São classes que não podem ser instanciadas diretamente (não podemos criar objetos delas). Podem conter métodos abstratos (métodos sem implementação) e métodos concretos (com implementação). Métodos abstratos são declarados com a palavra-chave abstract e devem ser implementados pelas subclasses. Classes abstratas servem como modelos para subclasses, definindo um contrato comum.

Interfaces: São como 'contratos' que definem um conjunto de métodos que uma classe deve implementar. São declaradas com a palavra-chave interface. Todos os métodos em uma interface são implicitamente public e abstract (até Java 8) Uma classe pode implementar múltiplas interfaces.

Exemplos de Código

                    
    // Classe Abstrata
    abstract class InstrumentoMusical {

        String nome;

        public InstrumentoMusical(String nome) {
        this.nome = nome;
        }
        
    abstract void tocar(); // Método abstrato
        void afinar() {

        System.out.println("Afinando o instrumento."); // Método concreto
         }
        }

// Subclasse concreta
class Violino extends InstrumentoMusical {

    public Violino(String nome) {
    super(nome);
    }

    @Override
    void tocar() {
    System.out.println("Violino tocando uma melodia.");
        }
    }

// Interface
interface Barulhento {

    void fazerBarulhoAlto();

}

// Classe que implementa a interface
class Bateria implements Barulhento {

    @Override
    public void fazerBarulhoAlto() {

    System.out.println("Bateria fazendo um som ensurdecedor!");
    }
}

public class ExemploAbstracao {

    public static void main(String[] args) {

    InstrumentoMusical meuViolino = new Violino("Stradivarius");

    meuViolino.tocar();

    meuViolino.afinar();

    Bateria minhaBateria = new Bateria();

    minhaBateria.fazerBarulhoAlto();

        }
    }

Exercícios:

  1. Modele um sistema de pagamento usando uma interface Pagavel com um método calcularPagamento(). Crie classes para diferentes tipos de pagamento (ex: Funcionario, Boleto, Servico) que implementam essa interface.

  2. Crie uma classe abstrata Veiculo com métodos para ligar(), desligar() e um método abstrato acelerar(). Crie subclasses Carro e Moto que herdam de Veiculo e implementam o método acelerar() de forma específica.
