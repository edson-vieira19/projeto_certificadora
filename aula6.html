Polimorfismo

Polimorfismo (do grego, 'muitas formas') é a capacidade de um objeto assumir diferentes formas ou tipos. Em POO, isso significa que um objeto pode ser referenciado por uma variável de um tipo mais genérico (superclasse) e se comportar de acordo com seu tipo específico (subclasse).

O polimorfismo permite escrever código mais flexível e genérico, pois podemos tratar objetos de diferentes classes de forma uniforme

Em Java, o polimorfismo é alcançado através de herança e interfaces.

Exemplos de Código

                    
// (Classes Animal, Cachorro e Gato definidas no exemplo de Herança)
    public class ExemploPolimorfismo {
                        
        public static void main(String[] args) {

        Animal animal1 = new Cachorro("Rex", "Pastor Alemão"); // Polimorfismo: Referência de superclasse

        Animal animal2 = new Gato("Felix"); // Polimorfismo: Referência de superclasse

        animal1.fazerBarulho(); // Saída: Cachorro late: Au au! (Comportamento de Cachorro)

        animal2.fazerBarulho(); // Saída: Gato mia: Miau! (Comportamento de Gato)

// Array de animais (polimorfismo em ação)

        Animal[] animais = new Animal[2];

        animais[0] = rex;
        animais[1] = felix;

        for (Animal animal : animais) {
        animal.fazerBarulho(); // Chamada polimórfica: cada animal faz seu próprio barulho
            }
        }
    }
