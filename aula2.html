Classes e Objetos

Em POO, tudo gira em torno de objetos e classes. Uma classe é como um modelo ou um molde que define a estrutura e o comportamento de um tipo específico de objeto.

Pense em uma classe como a planta de uma casa: ela especifica os cômodos, o layout e as características gerais.

Um objeto, por outro lado, é uma instância específica de uma classe. É uma casa construída a partir da planta. Assim como podemos construir várias casas a partir da mesma planta, podemos criar múltiplos objetos a partir da mesma classe.

Exemplo do mundo real

Classe: Carro

Objetos: Carro A (um Sedan vermelho), Carro B (um SUV azul), Carro C (um Hatch preto)

Cada objeto tem seu próprio estado (atributos) e comportamento (métodos), mas todos compartilham as características definidas pela sua classe.

Exemplo de código

// Definição da classe Carro
class Carro {
    String modelo;
    String cor;
    int ano;

    // Método para exibir informações do carro
    void exibirInformacoes() {
        System.out.println("Modelo: " + modelo + ", Cor: " + cor + ", Ano: " + ano);
    }
}

public class ExemploObjetosClasses {
    public static void main(String[] args) {
        // Criação de objetos da classe Carro
        Carro carro1 = new Carro();
        carro1.modelo = "Sedan";
        carro1.cor = "Vermelho";
        carro1.ano = 2023;

        Carro carro2 = new Carro();
        carro2.modelo = "SUV";
        carro2.cor = "Azul";
        carro2.ano = 2024;

        // Chamada do método para exibir informações
        carro1.exibirInformacoes(); // Saída: Modelo: Sedan, Cor: Vermelho, Ano: 2023
        carro2.exibirInformacoes(); // Saída: Modelo: SUV, Cor: Azul, Ano: 2024
    }
}
